# Smoketape Social Hooks

Grounded draft copy for public promotion. Pair these with
`examples/report-demo.sh` so readers can reproduce the report artifacts.

## Short Hooks

- Smoketape turns a readable YAML smoke test into Markdown and JSON proof
  reports for CLI workflows.
- The basic demo runs in a local temp sandbox and records assertion results that
  can be pasted into a PR or release checklist.
- Useful angle: replace opaque shell smoke scripts with tapes that explain what
  command ran and what output or files were expected.

## Video Brief

Show `examples/basic/smoketape.yml`, run `bash examples/report-demo.sh`, then
open `.demo-output/basic-smoke.md`. Close on the JSON report as the automation
entry point for agents and CI jobs.
